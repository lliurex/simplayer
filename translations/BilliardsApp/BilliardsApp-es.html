Acerca de:

Simulación basada en la física de bolas de billar 2D, usando el Motor de Física de Cuerpo Rígido 2D. También puede considerar esto como discos de hockey de aire sobre una mesa.

Haga clic cerca de un objeto para ejercer una fuerza con el mouse. Intente cambiar la gravedad, la elasticidad (rebote) y la amortiguación (fricción).

Experimentos para probar:

  • Compruebe si se conserva la energía: active "Mostrar energía". Tendrás que configurar amortiguación=0 y elasticidad=1.
  • Intente usar diferentes métodos de colisión, como "simultáneo". es la simulación sigue siendo realista?

Tenga en cuenta que esta es una simulación 2D y no trata aspectos importantes del mundo real bolas de billar como fricción y rotación o giro. Las bolas no giran en este simulación porque la fricción del borde no se calcula, por lo que las colisiones son esencialmente sin fricción.

El método de manejo de colisiones para este la simulación es importante. Usando el método "simultáneo" se obtienen valores simétricos y resultados no realistas. El artículo vinculado analiza por qué diferentes métodos de se necesita calcular el resultado de las colisiones.

También están disponibles: código fuente, documentación y un versión compilada simple que es más personalizable.
