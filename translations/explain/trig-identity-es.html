Identidades trigonométricas para cos(t) + sin(t)

Esta identidad trigonométrica muestra que una combinación de funciones de seno y coseno se puede escribir como una sola función de seno con un cambio de fase.

$$a \cos{t} + b \sin{t} = \sqrt{a^2 + b^2} \; \sin(t + \tan^{-1} \frac{a}{b}) $$

for b ≠ 0 and − π⁄2 < tan−1 a⁄b < π⁄2

(Tenga en cuenta que tan−1 significa arctan .) El cambio de fase es la cantidad tan−1 a⁄b , tiene el efecto de desplazar el gráfico de la función seno hacia la izquierda o hacia la derecha.

Para derivar esta identidad trigonométrica, suponemos que la combinación a cos(t) + b sin(t) se puede escribir en la forma c sin(K + t) para constantes desconocidas c, K .

a cos(t) + b sin(t) = c sin(K + t)
a cos(t) + b sin(t) = c sin(K) cos(t) + c cos(K) sin(t)

Usamos la fórmula para el seno de una suma de ángulos para expandir el lado derecho de arriba. Tener igualdad para cualquier valor de t , los coeficientes de cos(t) y sin(t) deben ser igual en los lados izquierdo y derecho de la ecuación.

a = c sin(K)
b = c cos(K)

Resolver este sistema de ecuaciones simultáneas nos lleva a

c = ± √(a2 + b2)
K = tan−1 a⁄b

Así que la identidad trigonométrica para b ≠ 0 es

a cos(t) + b sin(t) = ± √(a2+b2) sin(t + tan−1 a⁄b)

Si limitamos el arctan a estar dentro

− π⁄2 < tan−1 a⁄b < π⁄2

entonces siempre podemos elegir el + delante de la raíz cuadrada.
