Aquí hay algunos enlaces a otros sitios web de simulación física, además de sitios web generales de matemáticas y física.

  • PhysicsWeb.org tiene una gran lista de subprogramas de física.
  • Paul Falstad tiene una excelente colección de subprogramas de matemáticas y física.
  • Recursos de física The-Science-Lab es un directorio de sitios web relacionados con la física.
  • ExploreScience.com tiene varias buenas simulaciones de física, creadas con Shockwave.
  • mathengine proporciona software comercial para simulaciones de física utilizadas en videojuegos, ingeniería y académicos.

Enlaces matemáticos

  • El mundo de las matemáticas de Eric Weisstein es una extensa enciclopedia de información.
  • mathpuzzle.com tiene matemáticas recreativas divertidas.

Enlaces de física

  • Bad Movie Physics expone lo que está mal con todas esas escenas de choques y explosiones.
  • HyperPhysics "es un entorno de exploración de conceptos en física que emplea mapas conceptuales y otros enlaces estrategias para facilitar una navegación fluida".
  • Mecánica lagrangiana y hamiltoniana una introducción de Marek Rychlik.
