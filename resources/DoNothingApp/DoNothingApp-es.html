Acerca de

Simulación basada en la física de un trammel de Arquímedes, también conocida como "máquina de no hacer nada" o "molinillo de no hacer nada". Utiliza el Motor de Física de Cuerpo Rígido 2D.

Haga clic cerca del mango con el ratón para aplicar una fuerza al muelle. Active el "bloqueo extra" para ver el mecanismo de choque con otro bloque. Intente cambiar la elasticidad o la amortiguación. Colocar "fuerza del mango" para aplicar una fuerza constante al mango. Establezca la "tasa de rotación forzada" en cero para que el mecanismo alcance una alta tasa de velocidad. Active "Mostrar fuerzas" para ver el varias fuerzas en el trabajo.

Also available are: open source code, documentation and a simple-compiled version which is more customizable.
